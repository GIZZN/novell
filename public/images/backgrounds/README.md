# Фоны для локаций

Поместите сюда фоновые изображения для локаций:

## Требуемые файлы:

1. **park.jpg** - Городской парк 'Зелёная роща'
   - Раннее утро, солнечные лучи сквозь листву
   - Деревья, трава, тропинки
   - Используется в: scene_1, scene_1b, scene_2, scene_3_alex, scene_4_alex_1, scene_5_alex_decline, scene_6_alex_bad

2. **park_bench.jpg** - Скамейка у фонтана
   - Тихое место в парке
   - Скамейка, фонтан, утки
   - Используется в: scene_3_maya, scene_4_maya_1, scene_5_maya_decline

3. **cafe.jpg** - Кофейня 'Уголок'
   - Уютное заведение
   - Мягкие кресла, столики у окна
   - Джазовая атмосфера
   - Используется в: scene_5_alex_cafe, scene_6_alex_good

4. **library.jpg** - Городская библиотека имени Пушкина
   - Высокие потолки
   - Деревянные стеллажи с книгами
   - Настольные лампы, приглушённый свет
   - Используется в: scene_5_maya_library, scene_6_maya_good, scene_6_maya_bad

## Технические требования:

- **Формат:** JPG или PNG
- **Разрешение:** 1920x1080 (Full HD)
- **Соотношение сторон:** 16:9
- **Размер файла:** до 500KB (оптимизировать для веба)
- **Стиль:** Реалистичный или арт, единый стиль для всех фонов
